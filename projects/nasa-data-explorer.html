NASA Data Explorer

A NASA-inspired interactive web app combining space imagery and data through structured storytelling, responsive layouts, and clear visual hierarchy.

  • Data Visualization
  • Responsive UI
  • Visual Hierarchy
  • Storytelling

Overview

NASA Data Explorer brings publicly available space data to life through an interface built for curiosity. The project combined imagery from NASA's open APIs with structured content blocks to create an immersive browsing experience that felt scientific yet approachable.

The layout uses strong typographic hierarchy and intentional whitespace to guide users through dense data without overwhelming them, while the responsive design ensures the experience holds up on any screen size.

My Role

Interactive design and front-end development: visual system design, API integration planning, responsive HTML/CSS build, and user flow mapping.
