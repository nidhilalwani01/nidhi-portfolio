Dashboard UI  ·  2024

Intel Summit Check-In Dashboard

An interactive event check-in dashboard that tracks attendance, surfaces team analytics, and visualizes live progress for faster decisions.

  • Dashboard UI
  • Attendance Tracking
  • Team Analytics
  • Data Visualization

Overview

The Intel Sustainability Summit needed a real-time check-in solution that gave event coordinators instant visibility into attendance status, team breakdown, and capacity metrics. The dashboard was designed to surface the most important numbers at a glance, with drill-down views for deeper investigation.

Data visualizations were chosen to be immediately scannable under time pressure, using colour, size, and labelling to communicate status without requiring any prior training from the user.

My Role

UI design and front-end implementation: component design, data visualisation choices, responsive layout, and handoff documentation for the development team.
