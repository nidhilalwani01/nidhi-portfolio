Webpage Design  ·  2024

Intel Sustainability Timeline

An interactive sustainability storytelling webpage built with responsive structure, accessibility improvements, and RTL-ready localization support.

  • Semantic HTML
  • Accessibility
  • Responsive Design
  • RTL Localization

Overview

This project translated Intel's sustainability milestones into a scrollable, visually engaging timeline. The goal was to present complex data in a way that felt approachable for a broad audience, while meeting WCAG accessibility standards and supporting right-to-left reading layouts for global audiences.

Careful attention was given to semantic HTML structure, meaningful heading hierarchy, and keyboard navigability so that every user, regardless of ability, could follow the story.

My Role

Front-end design and development: information architecture, HTML/CSS build, accessibility audit, and localization implementation for RTL languages.
