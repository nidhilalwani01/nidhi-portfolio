Drop for Change

A charity: water-inspired browser game built with HTML, CSS, and JavaScript featuring scoring, weather interactions, and engagement-focused responsive design.

  • HTML/CSS/JavaScript
  • Game Mechanics
  • Responsive Design
  • Engagement UX
  • Gamification

Overview

Drop for Change is a browser-based game designed to raise awareness of the global clean water crisis. Inspired by charity: water's mission, the game uses simple drag-and-drop mechanics and a live scoring system to keep players engaged while communicating real impact statistics.

Weather interaction layers add a dynamic challenge — players must adapt their strategy as conditions change, creating re-playability and tying the gameplay experience directly to the real-world variability of water access.

My Role

Concept, design, and front-end development: game mechanic design, JavaScript logic, responsive CSS layout, and engagement UX strategy.
