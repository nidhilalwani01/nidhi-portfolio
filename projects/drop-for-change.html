Overview

Drop for Change is a web-based game concept designed to increase awareness of clean water access through playful, goal-driven interaction. The project blends product thinking, interaction design, and front-end implementation to make a social issue more memorable and participatory.

Problem

Awareness campaigns can struggle to sustain user attention when information is passive or text-heavy. The challenge was to create an experience that communicates purpose while keeping users engaged long enough to build understanding and intent.

Goals

  • Design an engaging game loop tied to clean water awareness.
  • Provide clear feedback so users understand progress and outcomes.
  • Keep controls and game states intuitive across device sizes.
  • Demonstrate end-to-end product and front-end execution.

Game Concept

The game frames clean water awareness through an action-based challenge where players collect or protect key resources while navigating changing environmental conditions. Progress, points, and challenge intensity work together to maintain focus and reinforce purpose.

Gameplay Mechanics

  • Scoring System: Rewards task completion and sustained performance.
  • Collision Detection: Adds challenge and consequence for mistimed actions.
  • Weather Events: Storms and lightning introduce dynamic difficulty changes.
  • Progress States: Signals round status and encourages replay behavior.

UX and Visual Feedback Decisions

  • Used clear score and status placement for quick interpretation.
  • Added visual feedback for hits, misses, and environmental hazards.
  • Designed readable contrast and hierarchy to reduce gameplay confusion.
  • Balanced visual energy with clarity to support both fun and comprehension.

Implementation Highlights

  • Built with semantic HTML structure and modular CSS styling.
  • Implemented gameplay logic in JavaScript for real-time interactions.
  • Integrated event-based updates for weather states and lightning hazards.
  • Adjusted layout and control behavior for responsive gameplay support.

Final Outcome

The final concept demonstrates how a browser game can translate awareness into active participation. Through interactive mechanics, responsive behavior, and clear feedback systems, the experience supports engagement while maintaining a strong educational message.

Reflections

  • Gameplay is most effective when feedback is immediate and meaningful.
  • Dynamic difficulty increases engagement when it remains understandable.
  • Product framing helps game interactions connect to real-world purpose.
  • A next iteration could include deeper narrative progression and level variety.
