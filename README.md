# nidhi-portfolio
Personal portfolio website showcasing UX, product design, and front-end development projects.

## Scaling This Portfolio

Use the project system guide for repeatable structure, naming, and maintenance:

- See `PORTFOLIO_SYSTEM.md`

Quick workflow for a new project:

1. Duplicate `projects/project-template.html` and rename using kebab-case.
2. Add media assets with consistent naming in `img/projects/<project-slug>/`.
3. Add one new project card in `index.html` using the commented card block.
4. Fill only existing links with `data-url` so optional buttons stay clean.
